#   URL app example
C++ & CMake project example using [curl][curl-link].

##  Requirements
- To build the project:
  - C++ compiler and build system
  - CMake
  - Git
- To build curl & OpenSSL:
  - make
  - autoconf
  - automake
  - libtool
  - GNU m4
  - nroff
  - perl

[curl-link]: https://github.com/curl/curl
